La programacion orientada a objetos es la forma en que java modela entidades del mundo real como por ejemplo cliente, producto, pedido y usuario, etc.

Todos estos objetos tienen estados (atributos) y comportamientos (metodos).

Encapsulamiento: Consiste en proteger el estado interno del objeto

  # Mala practica
    public class Cuenta {
        public double saldo;
    }

    Desde el main.java cualquiera podria modificar nuestro atributo
    cuenta.saldo = -1231321;

  # Buena practica
  public class Cuenta {
      private double saldo;

      public void depositar(double monto) {
          saldo += monto;
      }

      public double getSaldo() {
          return saldo;
      }
  }

  Ahora el objeto controla su estado
